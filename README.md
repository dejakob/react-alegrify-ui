# React Alegrify UI

React lib based on [Alegrify UI](https://github.com/dejakob/alegrify-ui).

Check out the live preview / storybook [here](https://dejakob.github.io/react-alegrify-ui)

## Typography
```jsx
<H1>Head 1</H1>
```
```jsx
<H2>Head 2</H2>
```
```jsx
<H3>Head 3</H3>
```
```jsx
<P>
    Paragraph
    <Em>Italic text</Em>
    <Strong>Bold text</Strong>
</P>
```
