# Alert

import Alert from '../lib/alert';

Use an alert to show a small notification.

    <Alert
        open
    >
        Hi, just to let you know
    </Alert>

## Properties

| Property | Type    | Required? | Default |
|----------|---------|-----------|---------|
|open      |Boolean  |No         |false    |
|className |String   |No         |''       |
